È il **descrittore** del file nel [[File System di UNIX]]
Tra gli attributi nell'i-node ci sono:
- **Tipo** di file (ordinario, direttorio, file speciale)
- **Proprietario, gruppo** (user-id, group-id)
- **Dimensione**
- **Data**
- **12 bit di [[File System#Liste di accesso e gruppi (es. UNIX)|protezione]]** 
- Numero di **links**
- **13-15 indirizzi di blocchi**, in base alla realizzazione
